## 05. 비트 조작

### 산술 우측 시프트 vs. 논리 우측 시프트

### 기본적인 비트 조작

#### 비트값 확인

```java
boolean getBit(int num, int i) {
    return ((num & (1 << i)) != 0);
}
```

#### 비트값 채워넣기

```java
boolean setBit(int num, int i) {
    return num | (1 << i);
}
```

#### 비트값 삭제하기

```java
boolean clearBit(int num, int i) {
    int mask = ~(1 << i);
    return num & mask;
}
```

#### 비트값 바꾸기

```java
boolean updateBit(int num, int i, boolean bitIs1) {
    int value = bitIs1 ? 1 l : 0;
    int mask = ~(1 << i);
    return (num & mask) | (value << i);
}
```
